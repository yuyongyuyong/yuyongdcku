# yuyongdcku

这是我的第一个 GitHub 仓库，用于学习代码管理、沉淀个人创作与量化交易相关的内容。

## 📁 仓库内容
- `README.md`: 仓库说明文档
- `短视频脚本`: 用于AI短剧/抖音视频的分镜脚本

## ✨ 功能
- 学习 Git 操作与版本管理
- 管理个人代码与创作资源
- 同步手机与电脑端的内容更新

## 🚀 使用说明
1.  克隆仓库到本地：`git clone https://github.com/yuyongyuyong/yuyongdcku.git`
2.  进入目录：`cd yuyongdcku`
3.  修改文件后提交：
    ```bash
    git add .
    git commit -m "更新内容"
    git push origin main
    ```
